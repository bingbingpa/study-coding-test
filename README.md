### String
- charAt(), int num = str.charAt(i) - '0';
- substring()
    - substring(beginIndex, endIndex)  시작인덱스 부터 마지막 인덱스 -1 까지
    - substring(index) 시작 인텍스부터 모두
- indexOf()
    - indexOf(String str) 문자의 위치를 리턴, 없으면 -1
    - indexOf(String str, int fromIndex) fromIndex 번째의 str 위치를 리턴
